# Getting Started BemCombinado

#### 1. Install Bower globally

```sh
$ npm install -g bower
````


#### 2. Install NPM dependencies

```sh
$ npm install
```

#### 3. Install Bower dependencies

```sh
$ bower install
```

#### 4. Type gulp and see the magic happens

```sh
$ gulp
```



OR ----
Just install bower globally and type:
```sh
$ npm start
```
